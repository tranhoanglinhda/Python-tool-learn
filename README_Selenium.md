Hướng dẫn nhanh để chạy `open_google_selenium.py`:

1) Cài Selenium:

```powershell
py -m pip install selenium
```

2) Chạy script:

```powershell
py open_google_selenium.py
```

Ghi chú: Selenium 4.10+ dùng Selenium Manager để tự động xử lý driver (Chrome/Edge/Firefox). Nếu gặp lỗi driver, cài `webdriver-manager` hoặc chỉ rõ `Service` với đường dẫn driver.
